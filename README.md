# Code Beacon Lab Dev Community Project

> **Code Beacon Lab** 스터디에서 진행하는 **Front-End** 프로젝트입니다.
> <br/>[Main Repository](https://github.com/code-beacon-lab/cbl-bis-fe)

<br/>

## 실행 환경
- **`Node`  16.16.0**
- **`npm`  9.2.0**
<br/><br/>

###### 실행 전 설치
```shell
npm install
```

<br/>

###### 실행 방법
```shell
npm run start
```
<br/>
